⚡ Por qué RED

Todo lo que necesitas.
Nada que no quieras.

Construido sobre principios de privacidad radical y descentralización real — sin compromisos.

🔐

Cifrado Total E2E

X25519 + AES-256-GCM con Double Ratchet y Perfect Forward Secrecy. Cada mensaje usa claves únicas de sesión. Ni siquiera nosotros podemos leer tus mensajes.

🌐

Red P2P Pura

Sin servidores centrales. Protocolo Gossip + DHT Kademlia + Onion Routing de 3 saltos. Imposible de censurar o bloquear por un solo actor.

🕵️

Anonimato Real

Identidades did:red:xxxxxxx efímeras generadas localmente. Sin número de teléfono, sin email, sin metadatos expuestos. Tu identidad cambia cada 24h automáticamente.

📡

Funciona Sin Internet

Bluetooth Low Energy y WiFi Direct permiten comunicación directa entre dispositivos cercanos. Forma una red mesh local y retransmite mensajes automáticamente.

📱

Multi-Plataforma

Android, iOS, Web y PWA desde un solo codebase (Next.js + Capacitor). Notificaciones push, modo offline, soporte multi-dispositivo con vinculación QR.

🔓

100% Open Source

Código auditado públicamente. Sin backdoors posibles. Auditoría de seguridad independiente. Fork libremente, contribuye o despliega tu propio nodo.

📡 Modo Offline

Sin internet.
Sin problema.

cuando se corta la red, cuando el gobierno bloquea Internet, cuando estás en una zona sin cobertura — RED sigue funcionando. Forma redes mesh locales en tiempo real usando el hardware que ya tienes.

📶
Bluetooth Low Energy (BLE)
Descubrimiento y mensajería directa en un radio de ~100 metros. Servicio GATT personalizado con cifrado RED nativo. Ideal para comunicaciones cortas y seguras.
📡
WiFi Direct / LAN Local
WebRTC peer-to-peer en red local (WiFi, hotspot, cable). Sin servidor STUN externo. Hasta 1 km de alcance con antenas externas. Alta velocidad para archivos y video.
🕸️
Protocolo Mesh Store-and-Forward
Los mensajes se almacenan y retransmiten automáticamente entre nodos. Si el destinatario no está cerca, el mensaje viaja de dispositivo en dispositivo hasta llegar.

🏗️ Arquitectura

Construido para
durar eternamente.

Cinco capas independientes, cada una con su propia responsabilidad de seguridad.

📱
Capa de Aplicación
Interfaces nativas para Android, iOS, Web y PWA. Soporte completo de chat, llamadas WebRTC, stories y todos los features de mensajería.
Next.js Capacitor TypeScript
🔒
Capa de Protocolo
Chat P2P, grupos descentralizados, llamadas WebRTC, stories y encuestas. Sin coordinación de servidor central.
WebRTC BLE WiFi Direct
🕵️
Capa de Identidad
DIDs efímeros, X25519 Diffie-Hellman con renegociación, Ed25519 para firmas, Perfect Forward Secrecy y deniabilidad plausible.
X25519 Ed25519 DID
🌐
Capa de Red
Gossip Protocol, DHT Kademlia para descubrimiento de peers, Onion Routing de 3 saltos para ocultación de metadatos. Red mesh local BLE/WiFi.
Gossip Kademlia DHT Onion
⛓️
Capa de Blockchain
Blockchain minimalista solo para registro de identidades DID. Consenso Proof of Stake. No almacena mensajes. Bloques cada 6 segundos.
PoS Rust ZK Proofs
